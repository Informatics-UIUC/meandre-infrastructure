In order to compile the infrastructure you need:

* Scala 2.7.6 or higher
* Mongo DB Java driver 1.2 or higher  (http://github.com/mongodb/mongo-java-driver/downloads)
* Crochet 0.1.4 or higher (http://github.com/xllora/Crochet/downloads)
* Snare 0.4.1 or higher (http://github.com/xllora/Snare/downloads)
* Jena 2.6.2 or higher (http://jena.sourceforge.net/downloads.html)
* Joda 1.6 or higher (http://joda-time.sourceforge.net/)
* Specs 1.6.2 or higher (http://code.google.com/p/specs/downloads/list)

In order to run the compiled infrastructure you will need to run
at least one instance of MongoDB. MongoDB can be downloaded from

* MongoDB http://www.mongodb.org/display/DOCS/Downloads (stable recommended)
